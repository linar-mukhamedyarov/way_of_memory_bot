# VK Bot "Голос памяти"
🏛️Вк-бот (библиотека vkbottle) для проведения тестов по историческим деятелям для проекта "Голос памяти" на конкурс PMCup.
Спроектирован как легковесное и расширяемое решение: новые тесты добавляются простым редактированием JSON-конфига без правки кода.
✨ Особенности
- JSON-driven: Весь контент (вопросы, варианты ответов, пояснения) хранится в tests.json.
- Гибкая архитектура: Легко добавлять новые команды и состояния благодаря модульности vkbottle.
- Масштабируемость: Код разделен на логику обработки и работу с данными.
- Обратная связь: После каждого ответа бот может выводить краткую справку об историческом деятеле.
